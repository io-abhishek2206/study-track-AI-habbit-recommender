StudyTrack AI – Student Performance Prediction & Dashboard

StudyTrack AI is an intelligent, data-driven system designed to analyze students’ study habits and predict academic performance using machine learning techniques. The project integrates data preprocessing, predictive modeling, clustering, and an interactive dashboard to deliver actionable insights and personalized recommendations for students.

⸻

Features
	•	Academic performance prediction using machine learning models
	•	Clustering of students based on study habits and behavioral patterns
	•	Data cleaning and preprocessing pipeline
	•	Interactive visualizations and analytics dashboard
	•	Personalized habit-based recommendations
	•	User-friendly web interface built with Streamlit

⸻

Project Workflow
	1.	Data collection
	2.	Data cleaning and preprocessing
	3.	Exploratory data analysis (EDA)
	4.	Feature engineering
	5.	Model training
	6.	Clustering
	7.	Visualization and prediction

⸻

Technologies Used
	•	Programming Language: Python
	•	Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
	•	Framework: Streamlit
	•	Machine Learning Techniques: Regression, K-Means Clustering

⸻

Dashboard Overview

The dashboard provides predicted academic scores, habit trend analysis, cluster-based insights, and performance comparisons over time through interactive charts and visualizations.

⸻

Model Details
	•	Supervised learning models are used for academic performance prediction
	•	Unsupervised learning models are applied for behavioral clustering
	•	Models are evaluated using accuracy measures and error analysis

⸻

Data Privacy
	•	No personally identifiable information is stored
	•	Data is used strictly for academic analysis and prediction
	•	The system follows basic data privacy and security principles
